Information

    This installation / update routine downloads all necessary application files for the MailBeez system and installs / updates them into /mailhive.

    Please see the MailBeez installer help-page for more information.

    As usual make sure to back up your Database using e.g. mysqldumper.net.

    The installer/updater creates a backup of /mailhive and stores it as zip-archive into /mailhive/cloudbeez/backup.

    All files and folders below /mailhive must be writeable - the installer checks this before updating the files. It will only update files, when all files are writable.
